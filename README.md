# Today's Scent — 오늘의 당신을 향으로 읽다

향을 한 번도 직접 묻지 않는 24개의 감각·장면 질문에 답하면, 7개 향 축(프레시·웜 / 라이트·인텐스 / 시트러스·우디 / 클래식·모던 / 데이·나이트 / 드라이·스위트 / 클린·스킨)에 점수가 누적되어 **마이클 에드워즈 프래그런스 휠** 기반 18개 향 계열 중 지금의 당신에게 가장 가까운 향을 찾아줍니다.

- **인다이렉트 진단** — "우디 좋아요?"가 아니라 장면·취향으로 묻고 뒤에서 향 축에 매핑
- **다축 z-score 매칭 + 신뢰도** — 1·2위 격차가 작으면 하이브리드(A×B)로 표현
- **오늘의 향 프레이밍** — 날짜 스탬프 + "3개월 뒤 다시"
- 단일 정적 `index.html` (데이터·엔진 분리, 외부 의존 없음). 폰트 Pretendard, 쿨 그레이 에디토리얼 무드.

로컬: `python3 -m http.server 8123` → http://localhost:8123
